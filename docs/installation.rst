============
Installation
============


Stable release
--------------

To install search_word_files, run this command in your terminal:

.. code-block:: console

    $ pip install search_word_files

This is the preferred method to install search_word_files, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for search_word_files can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/William-Lake/search_word_files

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/William-Lake/search_word_files/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/William-Lake/search_word_files
.. _tarball: https://github.com/William-Lake/search_word_files/tarball/master
